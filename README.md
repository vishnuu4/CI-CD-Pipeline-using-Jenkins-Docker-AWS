# CI-CD-Pipeline-using-Jenkins-Docker-AWS

## 📌 Overview
This project demonstrates a simple CI/CD pipeline using Jenkins, Docker, and AWS EC2. Jenkins automates the build and deployment process whenever code is pushed to GitHub.

---

## 🛠️ Tools Used
- Jenkins
- Docker
- AWS EC2
- GitHub
- Ubuntu Linux

---
## 📚 Learning Outcomes

- Jenkins Pipeline Creation
- Docker Image Management
- CI/CD Automation
- AWS EC2 Deployment
- GitHub Integration
- Linux Server Administration
---

## ⚙️ Workflow
1. Push code to GitHub
2. Jenkins pulls the latest code
3. Docker image is built
4. Container is deployed on AWS EC2
